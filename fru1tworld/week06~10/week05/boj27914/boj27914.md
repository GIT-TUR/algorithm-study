## 문제

[27914](https://www.acmicpc.net/problem/27914)

## 해결 방법

prefix sum으로 해결

```
for (int i = 2; i <= n; ++i) {
    if (num[i] != k) {
        contin++; // 연속 가능한 구간 길이 증가
        num[i] = num[i - 1] + contin; // 이전 값에 현재 가능한 구간 수를 추가
    } else {
        num[i] = num[i - 1]; // 학년이 `k`라면 구간 개수 변화 없음
        contin = 0; // 연속 길이를 초기화
    }
}
```
