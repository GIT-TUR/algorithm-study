## 관련문제

리트코드 394번 Decode String 문제

## 문제 풀이

stack 활용 문제
알파벳들은 들어가는 순서가 입력받은 순서와 동일
\*,/,+,-의 적절한 위치만 파악해서 넣어주면됨
설명이 어렵군...
")"가 들어오거나 "+", "-"가 들어오면 들어있는 모든 요소들을 빼서 answer에 추가해주면됨 ( 괄호 빼고 )
"_"나 "/"는 연관이 있는 "_"나 "/" 요소가 있기 전까지 빼서 answer에 추가해주면됨
마지막에 남아있는 요소들을 순회하며 answer에 추가
